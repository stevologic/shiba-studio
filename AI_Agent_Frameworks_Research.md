# Latest AI Agent Frameworks Research (2024-2025)

## Top Frameworks
1. **CrewAI** - Leading multi-agent team orchestration. Focus on role-based agents, tasks, and crews.
2. **AutoGen (Microsoft)** - Conversational multi-agent framework. Supports dynamic agent conversations and tool use.
3. **LangGraph (LangChain)** - Graph-based stateful workflows for reliable agent systems. Supports cycles, persistence, and human-in-loop.
4. **Transformers Agents 2.0 (Hugging Face)** - Code-first agent system using LLMs to orchestrate tools/models.
5. **OpenAI Swarm** - Lightweight multi-agent orchestration framework.
6. **LlamaIndex Workflows** - Event-driven agentic workflows.

## Trends
- Shift to multi-agent systems and orchestration.
- Graph/state machine approaches for reliability (LangGraph).
- Integration with tools, RAG, and skills.
- Focus on open-source and local-first agents.
- Agent skills marketplace emerging (NVIDIA, AWS).
