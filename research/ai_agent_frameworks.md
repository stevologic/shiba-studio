# Latest AI Agent Frameworks Research (2024-2025)

## Top Frameworks
- **CrewAI**: Role-based multi-agent teams, easy orchestration.
- **AutoGen (Microsoft)**: Multi-agent conversations, code execution.
- **LangGraph**: Stateful graphs for complex agent workflows.
- **OpenAI Swarm**: Lightweight multi-agent orchestration (experimental).
- **MetaGPT**: Multi-agent software dev simulation.
- **CAMEL**: Communicative agents for "mind" exploration.
- **LlamaIndex Agents**: Tool-calling + workflows for RAG.

## Trends
- Shift to multi-agent systems over single agents.
- Emphasis on evaluation, memory, and tool integration.
- Integration with LLMs like GPT-4o, Claude 3.5, Llama 3.1.
- Focus areas: Planning, reflection, collaboration.

Sources: GitHub trending, Lilian Weng blog, arXiv.
