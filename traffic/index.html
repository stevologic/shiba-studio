03 · PRIVACY-FRIENDLY ANALYTICS

What GoatCounter does

Open source · hosted or self-hosted
PRIVACY MODEL

No persistent browser identifiers

GoatCounter does not store IP addresses, full user-agent strings, cookies, localStorage identifiers, or cross-site profiles. Its default reports use separate aggregate tables that cannot be linked into a person’s browsing history.

NO COOKIES · NO PERSISTENT ID
USEFUL SIGNALS

Visits without surveillance

It reports visits, popular pages, referrers, browsers, operating systems, languages, screen widths, and approximate countries. Repeat loads of the same page during a session normally count as one visit.

SHIBA STUDIO SETUP

The API key stays local

Shiba Studio encrypts the GoatCounter API token on your machine, uses it to read dashboard statistics, and installs the lightweight counting script on the public Pages files. The secret is never published to this site.
